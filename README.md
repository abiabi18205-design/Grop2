## FITGUIDE - Your quick, smart fitness helper.

## Project Overview
The **AI Fitness Coach** is an interactive web application designed to help users achieve their fitness goals through guided workouts, nutrition advice, progress tracking, and form analysis. The project integrates features like a live chatbot, camera-based exercise form scanner, dark/light theme toggle, and responsive navigation to provide a complete fitness coaching experience online.



## Color Palette

| Color Role                  | Hex Code    | Usage Example                                    |
|------------------------------|------------|-------------------------------------------------|
| **Primary Orange / Accent**  | `#FF5500`  | Buttons, highlights, icons, headings           |
| **Dark Background / Card**   | `#FFFFFF`  | Main background, cards, scanner overlay        |
| **Light Background / Text**  | `#F5F5F5`  | Light mode background, general text            |
| **Secondary Gray**           | `#888888`  | Subtext, secondary buttons, inactive elements  |

## Features
- Fully responsive layout with mobile-friendly navigation
- Login form with simulated authentication and success notifications
- Start your fitness journey button with smooth page transition
- AI chatbot for workouts, nutrition, form, and progress guidance
- Camera scanner to analyze exercise form and provide real-time feedback
- Dark and light theme toggle with local storage support
- Progress bar simulation for page loading feedback
- Toast notifications for interactive messages
- Intersection observer animations for feature cards
- Animated gym posters for subtle UI enhancements
